Botibal

A silly, quizzical XMPP bot based on the `SleekXMPP`_ (2.7) and
`SliXMPP`_ (3.4+) Python libraries.

.. _SleekXMPP: https://github.com/fritzy/SleekXMPP
.. _SliXMPP: https://dev.louiz.org/projects/slixmpp
